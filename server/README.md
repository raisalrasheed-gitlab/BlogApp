# React + Vite

1. Clone the respository :
   **https://github.com/raisalrasheed-gitlab/BlogApp.git**
2. Navigate to the project directory :
   `$ cd server`
3. Install dependencies :
   `$ npm i or npm install`
4. Start the server :
   `$ npm run dev`
5. Open your browser and visit :
   **http://localhost:9001/**
